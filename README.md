# Star Battle

## 🧠 Descripción
Juego para 2 jugadores por turnos desarrollado en C++ que simula batallas en un tablero 10x10 con eventos aleatorios.

## 🎮 Características
- Tablero dinámico
- Eventos: naves, xenomorfos, yautjas, agujeros negros
- Sistema de combate por turnos
- Ranking de jugadores

## 🎮 Objetivo del juego
Uno de los 2 jugadores tendrá que destruir un total de 7 naves del otro jugador.
Las reglas estaran en el juego al iniciar cada partida.

## 🛠️ Tecnologías
- C++
- Consola (terminal)

## ▶️ Cómo ejecutar
```bash
g++ startBattle.cpp -o juego
./juego
